# Roadmap - MOBY Standards
---

## v0.1.0 - Framework

- Rust CLI initialized
- YAML-backed standards
- Weight normalization
- Category normalization
- Validation command
- JSON export command

## v0.2.0 - Product Type Aliases

- Add product-type aliases
- Normalize product types
- Validate product types against categories

## v0.3.0 - Package Size Standards

- Add package-size registry
- Distinguish flower weights, vape sizes, edible package sizes, concentrate sizes
- Add category-aware normalization

## v0.4.0 - Potency Units

- Standardize THC/CBD units
- Percent vs mg/g vs mg/package vs mg/serving
- Add potency field schema

## v0.5.0 - State Overrides

- Allow state-specific standard sets
- Example: NV flower package weights
- Example: MA package limits
- Example: OR category naming

## v0.6.0 - Source Mapping Metadata

- Add source system metadata
- Track raw input, normalized output, confidence, and source notes


## v0.7.0 - MOBY JSON Schema Export

- Export JSON Schema
- Export TypeScript types
- Export machine-readable standard references

## v1.0.0 - Canonical Cannabis Data Standard

- Stable standard keys
- Documented aliases
- State-aware extension support
- MOBY-compatible output contracts
